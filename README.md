# Dot Files for i3WM on Ubuntu
## by re4nightwing
Contains config files for,
- `I3`
- `I3blocks`
- `I3status`
- `I3lock`
- `Nvim`
- `Polybar`
- `Ranger`
- `Rofi`
- `Compton` 
- `Dunst`
